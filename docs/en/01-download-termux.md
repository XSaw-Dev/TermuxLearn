📥 Download Termux

Welcome to TermuxLearn!

In this chapter, you will learn how to download and install Termux properly.

«⚠️ Do not download Termux from the Google Play Store.

The Play Store version is outdated and no longer maintained.»

Method 1 — GitHub (Recommended)

Download the latest APK from the official Termux GitHub Releases page.

The GitHub version always provides the latest updates.

Method 2 — F-Droid

You can also install Termux from F-Droid.

This version is actively maintained and regularly updated.

Installation

1. Download the APK.
2. Open the APK file.
3. Allow installation if prompted.
4. Wait until the installation finishes.
5. Launch Termux.

Initial Update

Run the following command:

pkg update && pkg upgrade -y

Verify Installation

Run:

termux-info

If Termux information is displayed, the installation was successful.
